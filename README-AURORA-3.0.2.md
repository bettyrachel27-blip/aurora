# Aurora 3.0.2 — Routines intelligentes

## Nouveautés
- Menu ••• sur chaque habitude : valider, reporter, choisir une date ou supprimer.
- Éditeur réel pour les routines Travail, Repos, Récupération, Vacances et Repos santé.
- Sélection des habitudes incluses dans chaque routine.
- Ajout d'une nouvelle habitude directement dans une routine.
- Synchronisation cloud déclenchée après chaque modification.

## Déploiement
Remplacer les fichiers du dépôt GitHub par ceux de cette archive. Render redéploie automatiquement. Aucun changement n'est nécessaire dans Google Apps Script.
