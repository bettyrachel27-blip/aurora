# Aurora 3.0.1 — Agenda intelligent

## Nouveautés
- Menu ••• sur chaque rendez-vous
- Modifier un rendez-vous
- Reporter à demain, au prochain jour travaillé ou au prochain jour libre
- Choisir une nouvelle date
- Supprimer un rendez-vous
- Tri automatique par date puis heure
- Synchronisation cloud déclenchée après chaque action
- Fonctionnement identique sur mobile et ordinateur

## Déploiement
Remplacer les fichiers du dépôt GitHub par ceux de cette archive. Render redéploiera le site. Aucun changement Google Apps Script n’est requis.
